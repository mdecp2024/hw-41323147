w15

第一題

從 1 累加到 100

total = 0
for i in range(1, 101):
    total += i

print(f"1 到 100 的總和是：{total}")

第二題

設計一個函式 addto(1, 100), 可以完成上述的累加結果

def addto(start, end):
    """
    計算從 start 到 end 的累加總和
    """
    return sum(range(start, end + 1))  # 使用 sum 和 range 計算總和

# 測試函式
result = addto(1, 100)  # 計算 1 到 100 的總和
print(f"addto(1, 100) 的結果是：{result}")

第三題

設計一個函式 add_only_even(1, 100), 可以完成從第一個輸入整數, 累加到第二個輸入變數的累加結果

def add_only_even(start, end):
    """
    計算從 start 到 end 的偶數總和
    """
    return sum(i for i in range(start, end + 1) if i % 2 == 0)

# 測試函式
result = add_only_even(1, 100)  # 計算 1 到 100 的偶數總和
print(f"add_only_even(1, 100) 的結果是：{result}")



第四題

設計一個函式 add_avoid_8(1, 100), 可以完成從第一個輸入整數, 累加到第二個輸入變數的累加過程, 必須避開數字出現"8"的整數

def add_avoid_8(start, end):
    """
    計算從 start 到 end 的偶數總和，避開包含數字 '8' 的整數。
    """
    total = 0
    for i in range(start, end + 1):
        # 檢查是否為偶數並且不包含 '8'
        if i % 2 == 0 and '8' not in str(i):
            total += i
    return total

# 測試函式
result = add_avoid_8(1, 100)  # 計算 1 到 100 的偶數總和，避開包含 '8' 的偶數
print(f"add_avoid_8(1, 100) 的結果是：{result}")

心得:書寫四題過後，我發現不同的方法也可以做得出相同的答案，這讓我發覺不用強求一定的過程，條條大路通羅馬，這也適用於未來讀書，可以用不同的方法找出答案，不要太鑽牛角尖。
